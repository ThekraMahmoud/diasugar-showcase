# DiaSugar – Smart Diabetes Management Application

DiaSugar is an intelligent mobile application developed using Flutter and Firebase to support diabetes management through real-time monitoring, communication, and personalized healthcare features.

## Features

* Multi-role system (Doctor, Adult Patient, Child Patient, Admin)
* Real-time glucose monitoring
* Smart alerts and notifications
* Doctor-patient chat system
* Appointment management
* Child-friendly interactive interface
* Task and goal tracking
* Firebase real-time synchronization
* Admin dashboard and analytics

## Technologies Used

* Flutter
* Firebase Authentication
* Cloud Firestore
* Firebase Cloud Messaging
* Dart
* fl_chart
* Riverpod (currently improving the architecture)

## Project Overview

The application was developed as a graduation project through collaboration between the College of Engineering and Computer Science and the College of Pharmacy.

DiaSugar aims to enhance communication between patients and healthcare providers while encouraging healthier daily habits through a smart and user-friendly experience.
